# Minishell

## Description

Minishell is a simplified Unix shell implementation designed to mimic basic functionalities of common shells like `bash`. The goal of this project is to deepen understanding of:

- Process creation and management (`fork`, `execve`, `waitpid`)
- File descriptor manipulation (input/output redirection)
- Handling of environment variables
- Parsing and execution of user commands

The shell supports built-in commands, pipelines, and redirections, while providing an interactive prompt for command execution.
